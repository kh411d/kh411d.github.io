[NOTE] Upgrade NodeJS lokal via NPM dan "n" module

Khalid Adisendjaja · February 19, 2014

Sebelum NodeJS-nya di-upgrade, cek dulu versi yang ter-install,

$node -v
v0.10.12

Bersih2x NPM Cache,

sudo npm cache clean -f

Install “n” Node binary manager ,

sudo npm install -g n

Upgrade Node ke versi terakhir,

sudo n 0.10.26

Atau bisa langsung upgrade ke versi stable terakhir,

sudo n stable

Cek versi node yg telah terupgrade,

$node -v
v0.10.26
